Birdle

Professor Lydia Chilton

February 2020

Given as a project for a User Interface Design class taught by Lydia Chilton, the goal of this project was to teach about the kinds of birds that reside in Guam and the Marianas Islands. The user is able to find information on the variety of both endemic and ecdemic avians available in this easy to use database.
